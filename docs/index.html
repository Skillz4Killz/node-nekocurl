Nekocurl

A HTTP client class that uses plug-in drivers to do HTTP requests.

new Nekocurl(url: string, options: object): Nekocurl
Author: Charlotte Dunois (https://github.com/CharlotteDunois/node-nekocurl)
Parameters
url (string) The url, what else?
options (object = {}) Any options you want to pass.
Name Description
options.driver (string | undefined) The driver which should be used.
options.method string (default 'GET') The request method.
options.headers object (default {}) HTTP-Headers.
options.data (string | null) (default null) The request payload.
options.files array (default []) An array containing objects ({ name, data, filename }), each representing a file.
options.autoString boolean (default true) Automatically turn buffers into strings.
options.encoding (string | undefined) Encoding (only used by some drivers, e.g. request).
options.json boolean Set true, if payload is JSON and/or Nekocurl should automatically parse response JSON (Snekfetch does it depending on Content-Type).
Returns
